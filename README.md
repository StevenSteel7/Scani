# Scani

A fast macOS OCR utility that lets you capture, extract, copy, search, and translate text directly from your screen — without breaking your workflow.

Scani works system-wide. Hold a shortcut, select text on screen, and instantly interact with it.

---

## Features

### Core Scanning

* Global `⌘` hold hotkey — works from any app
* Configurable hold duration (`0.5s – 5.0s`)
* Apple Vision framework OCR on the active screen

### Overlay & Selection

* Auto frame detection around text clusters
* Free-draw selection box
* Drag, resize, and nudge (arrow keys) the frame
* Line-by-line token selection with adjustment handles
* Select All with `⌘A`

### Actions

* Copy (`⌘C`)
* Web Search (`Return`)
* Google Lens image search
* Apple Translate integration
* Editable text preview panel (`Space`)
* Word and character count

### Permissions & Onboarding

* Guided first-launch setup
* Live permission status indicators
* Direct links to System Settings
* Relaunch prompt after permissions are granted

### Licensing & Trial

* 10-day free trial
* License key activation
* Per-device seat management
* Server-side validation

### App Experience

* Menu bar app
* Launch at Login
* Frosted glass UI
* Smooth spring animations
* Retina display support
* Multi-display aware
